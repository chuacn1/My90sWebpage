Side Note: Writing HTML is all about the structure of your content. HTML is used to define the elements of your website, like headings, paragraphs, lists, and images, so that browsers know how to display the content. It's important to remember that HTML isn’t concerned with how things look—that’s what CSS (Cascading Style Sheets) is for. CSS is where you control the design and layout, such as colors, fonts, and spacing. When writing HTML, focus on organizing and structuring the content of your website. The appearance and style will come later with CSS.
